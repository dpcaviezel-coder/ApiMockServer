# API Mock Server

A lightweight C# mock API server designed for QA automation, frontend development, and backend prototyping. Create custom endpoints, simulate delays, inject errors, and serve JSON responses without needing a real backend.

## Features

### ✔ 1. GET Mocking
Serve static JSON responses from local files.

### ✔ 2. POST Mocking
Simulate resource creation with custom JSON bodies.

### ✔ 3. Auth Mocking
Mock login, token generation, and auth failures.

### ✔ 4. Configurable Routes
Add new routes using `routes.json` without touching code.

### ✔ 5. Delay Simulation
Simulate slow APIs with configurable delays.

### ✔ 6. Error Simulation
Inject 400, 401, 403, 404, and 500 errors.

### ✔ 7. Randomized Responses
Return different responses each time.

### ✔ 8. Logging
Log all requests with timestamps, headers, and bodies.

### ✔ 9. Custom Endpoints
Create new endpoints easily using simple C# classes.

### ✔ 10. Server Configuration
Control port, delay, error mode, logging, and more.

### ✔ 11. CLI Menu
Start/stop server, reload routes, toggle features.

### ✔ 12. JSON Validation
Ensure all responses are valid JSON.

### ✔ 13. Hot Reload
Reload routes without restarting the server.

### ✔ 14. Dynamic Response Builder
Build responses using templates and variables.

### ✔ 15. Request Inspector
View detailed request logs for debugging.
