# ApiMockServer

A lightweight C# mock API server for testing, prototyping, and QA automation.

## Features

- JSON-based route configuration
- JSON response templates
- Auth, users, products, and custom endpoints
- Delay simulation for latency testing
- Simple logging
- Easy to extend with new endpoints and responses

## Structure

- `Config/` — server and route configuration
- `Responses/` — JSON response bodies
- `Endpoints/` — endpoint handlers
- `Server/` — core server engine
- `Examples/` — usage examples
